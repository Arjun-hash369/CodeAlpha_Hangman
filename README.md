# 🎮 Hangman Game (Python)

This is a simple text-based Hangman Game built using Python as part of my internship task.

## 🚀 Features
- Random word selection
- 6 incorrect guesses allowed
- Real-time word display using underscores
- Input validation

## 🧠 Concepts Used
- random module
- while loop
- if-else conditions
- lists and strings
